# 📌 Project Overview

FinSight is an end-to-end financial analytics solution built using Power BI. It consolidates transaction-level financial data into interactive dashboards that help stakeholders monitor key financial metrics, customer behavior, transaction performance, and operational trends.

The dashboard provides an intuitive interface with dynamic filters, KPI cards, and drill-down analysis, enabling faster and more informed business decisions.

---

# 🎯 Business Objectives

- Monitor overall financial performance.
- Track transaction volume and total transaction value.
- Analyze fees and tax collections.
- Understand customer transaction behavior.
- Compare customer segments and occupations.
- Identify state-wise financial performance.
- Support business decision-making using interactive analytics.
